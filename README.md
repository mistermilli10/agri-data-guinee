# agri-data-guinee

Une plateforme de gestion de données agricoles en République de Guinée.

## Prompts de génération

Le fichier [`PROMPTS.md`](./PROMPTS.md) contient l'intégralité des consignes et prompts prêts à l'emploi pour générer automatiquement la plateforme **AgriData Guinée** à l'aide d'outils de génération de code (Codex, ChatGPT Code Interpreter, Hostinger Horizon, etc.).

Ces prompts regroupent :

- Un master prompt “one-shot” pour générer la totalité de l'application.
- Des prompts modulaires par page ou fonctionnalité.
- Le schéma de données Prisma, la description de l'API, les textes d'interface et les critères d'acceptation.
- Un rappel du script de seed attendu et un plan de déploiement (CI/CD, Vercel, Supabase, etc.).

Servez-vous-en pour initialiser rapidement le projet ou pour régénérer certains fichiers de manière ciblée.
